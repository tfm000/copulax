# \_src

All objects within this module are implemented in pure JAX and are designed to
work with JAX's grad and hessian functions.

## Subpackages

| Directory       | Description                                                                           |
| --------------- | ------------------------------------------------------------------------------------- |
| `univariate/`   | Univariate distribution implementations, fitting utilities, and goodness-of-fit tests |
| `multivariate/` | Multivariate distribution implementations and correlation/covariance utilities        |
| `copulas/`      | Copula distribution implementations (elliptical and Archimedean families)             |
